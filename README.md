# number-guessing-game
Preview Link: https://issho106.github.io/number-guessing-game/

How to play:
1. An evil alien has taken control of all information technology on earth.
2. To regain access to a device, some user needs to guess a number between 1-100 correctly for some device.
3. A user can leave at any time (give up) by entering anything which isn't a number
4. When a user wins back access to a device, he can go again and save another device from the evil alien.
